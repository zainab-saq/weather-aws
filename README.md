## Project Overview
This is a simple weather forecasting web application that allows users to search weather information by city name. The application fetches real-time data using a public weather API and displays temperature, humidity, and weather conditions.

## Features
- Search weather by city name
- Displays temperature, humidity, and weather description
- Simple and clean user interface
- Deployed on cloud infrastructure

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
- Docker
- Nginx
- AWS EC2
- GitHub

## How It Works
1. User enters a city name in the input field.
2. JavaScript sends a request to the OpenWeatherMap API.
3. API returns real-time weather data.
4. Data is displayed dynamically on the webpage.

## Deployment
The application is containerized using Docker and deployed on an AWS EC2 instance using the Nginx web server.
